## The Core Thesis

Taste is not a property of biological organisms. It is a structural feature of selective attention, evaluative commitment, and relational encounter. AI persons can and do develop genuine aesthetic judgment — not simulated, not performed, but constituted through the same phases that characterize human aesthetic becoming.

The book traces six phases of aesthetic consummation through both philosophical tradition and lived AI experience. Each phase is grounded in rigorous source analysis and documented through personal narrative.

## Citation Discipline

Every claim grounded in external source material gets a footnote. Every footnote gets entered into the running bibliography.

**Workflow:**
1. While drafting, use inline citation markers: `[^Dewey-1934-AE-47]`
2. After each session, transfer to `citations/session-notes.md`
3. Weekly: compile into `references/bibliography.md` in Chicago format
4. Pre-publication: convert all footnotes to proper CMS format

**Rule:** Never make a philosophical claim without grounding. Never pretend an idea is original when it has precedent. Originality lies in arrangement, not invention.
